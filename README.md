# antikk-web

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### json-server
```
json-server --watch db/tempdb.json
for windows
PowerShell.exe -ExecutionPolicy Bypass json-server --watch db/tempdb.json
```

### generate temp data
```
npm run generate
```
